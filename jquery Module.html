Q.1.What is jquery?
jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility.
Q.2.How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation ?
css() - Sets or returns the style attribute. addClass() - Adds one or more classes to the selected elements. removeClass() - Removes one or more classes from the selected elements. toggleClass() - Toggles between adding/removing classes from the selected elements.
Q.3.How to create slider with animation?
Make a container element to for the images after that add the necessary styling to position the images inside container.And then create an array to hold the image URLs and use jQuery to dynamically populate the container with the images.
